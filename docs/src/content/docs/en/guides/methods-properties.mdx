# Methods & Properties

After initialize Zoomist, you can use initialized instance in variable with methods and properties.

For example: 

```js
const zoomist = new Zoomist('.zoomist-container')

// method
zoomist.zoom(0.1)
// property
console.log(zoomist.transform)
```
