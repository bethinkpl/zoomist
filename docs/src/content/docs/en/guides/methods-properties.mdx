# Methods & Properties

After initialize Zoomist, you can access methods and properties in initialized instance.

For example: 

```js
const zoomist = new Zoomist('.zoomist-container')

// method
zoomist.zoom(0.1)
// property
console.log(zoomist.transform)
```
