# 事件

Zoomist 提供一些有用的事件讓你監聽。

事件能被以兩種方式被註冊。

1 . 在選項中以 `on` 註冊事件：

```js
new Zoomist('.zoomist-container', {
  on: {
    zoom(zoomist, scale) {
      console.log(zoomist, scale)
    }
  }
})
```

2 . 在實例化後使用方法 `on` 註冊事件：

```js
const zoomist = new Zoomist('.zoomist-container')

zoomist.on('zoom', (zoomist, scale) => {
  console.log(zoomist, scale)
})
```
